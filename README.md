## 📌 فكرة المشروع

يهدف هذا المشروع إلى تقديم تصميم احترافي لموقع ويب يعكس مهاراتي في واجهات المستخدم (Front-End).  
الموقع يتميز بـ:

- تصميم جذاب ومتجاوب (Responsive)
- تنظيم واضح للصفحات
- استخدام أيقونات وفونت عصري
- تجربة مستخدم مريحة

---

## 🛠️ الأدوات المستخدمة

- 💻 HTML5
- 🎨 CSS3
- 🌐 Font Awesome
- 🧩 تصميم بسيط ونظيف مناسب لجميع الشاشات
